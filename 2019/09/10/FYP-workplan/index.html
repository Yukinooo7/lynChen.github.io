Draft FYP work plan

FYP work plan

Our final target is design a real-time monitor application so we will work together to solve image processing algorithms and work on applicaiton individually. We decide to use alige method to develop this application. The first semester we will finish the algorithm part and software design including experimental images capturing. The second semester we intend to implement the application functions and test it.

Algorithm Part

Go to lab weekly and capture more experimental images.

Integration phase (Semester 1 WEEK 3 (THIS WEEK) starts)

  • Review bio-scaffold image process algorithms from SURF project (Week 3).
    • Fixed and adjust camera position
    • Run SURF image process algorithms and guarantee later printing and capturing process can run sucessfully.

      Adjustment and improvement phase (Semester 1 WEEK 4 - 10 (MAYBE EARLIER))

  • Adjust image process algorithms and capture more experimental pictures (During this phase, capture pictures weekly and adjust algorithms if needed)
  • Implement bio-scaffold image classification
    • Classification may depend on two values: entropy and contours, e.g. normal one looks like a rectangle, without abnormal fiber, Fiber one means contours is similar to rectangle but has some abnormal fiber, Abnormal means the contour is irregular shape and has too much abnormal fiber.
    • may consider other algorithms about bio-scaffold classification
  • Improve CNN learning rate (It becomes higher after we capture more experimental images or has better pre-process algorithms)
  • Implement robustness, maintaineous, low coupling and document (relative to later software part)

We will also improve the algorithms in later phases
